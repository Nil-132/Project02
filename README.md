# My PW Replica

A full-stack replica of Physics Wallah website with admin panel.

## Features
- Subject-wise chapters and lectures
- Admin panel (Add/Delete Chapter & Lecture)
- Manual completion marking (like real PW)
- Search and filters
- MongoDB backend

## Tech Stack
- Frontend: HTML, Tailwind CSS, Vanilla JS
- Backend: Node.js + Express
- Database: MongoDB

## How to Run
1. Clone the repo
2. `npm install`
3. Create `.env` file with your `MONGO_URI`
4. `npm run dev`
5. Open `http://localhost:3000/subjects.html`
